faucet
======

Simple colored-coins faucet for testnet


Install
===========

Starting at root of git repo.

    npm install

    cd node_modules/cc-wallet-engine/
    
    npm install
    
    grunt compile
    
    cp cc-wallet-engine.js ../../pure-clientside/
    
    cd ../../pure-clientside
    
    python -m SimpleHTTPServer
    
Open browser at localhost:8000
